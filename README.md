# 吴彦组的万能计算器
### 组名 吴彦组
### 组员 马龙 左丹阳
#### 项目演示地址 http://counter.marlon1475.top/

## 文件系统

caculator-web Vue项目（前端）<br />
<p>--public index主页面</p>
<p>--src 前端资源文件（代码）</p>
<p>----api 接口调用</p>
<p>----components 导航组件</p>
<p>----enums 枚举值统一处理</p>
<p>----mixins 状态管理</p>
<p>----router 路由管理</p>
<p>----store 全局变量管理</p>
<p>----view 页面组件</p>
<p>------Home.vue 全局主页组件</p>
<p>------CounterPage.vue 计算器及主页面组件</p>
<p>------EvaluationPage.vue 综测计算</p>
<p>------FuntionPage.vue 函数图像组件</p>
<p>------HealthyPage.vue 健康计算组件</p>
<p>------CommodityPage.vue 一周开销组件</p>
<p>----App.vue 主组件</p>
<p>----main.ts 全局引入包管理</p>
<br/>
caculator-server SpringBoot项目（后端）<br />
<p>--sql 数据库文件</p>
<p>--img 图像文件</p>
<p>--src 后端资源文件（代码）</p>
<p>----main 项目代码</p>
<p>------java/com.example.yin java代码</p>
<p>--------domain 实体类</p>
<p>--------dao 数据连接层（与数据库的交互部分）</p>
<p>--------service 业务层（核心业务代码）</p>
<p>--------controller 接口层（处理来自前端的请求）</p>
<p>------resource 全局配置文件</p>
<p>----main 项目代码</p>
<p>----test 测试代码</p>
<br/>
